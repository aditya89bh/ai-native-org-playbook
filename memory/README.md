# Memory Architecture

Memory is the layer that lets an AI-native organization learn from prior work.

This section covers:

- Memory event schemas
- Decision memory
- Meeting memory
- Project memory
- Customer memory
- Policy memory
- Incident memory
- Learning memory
- Retention policies
- Correction workflows
- Forgetting rules

## Design rule

Capture memory only when it improves future decisions, coordination, governance, or learning.
