# Workflow Library

Workflow playbooks show how specific business processes change in an AI-native organization.

Each workflow should include:

- Current workflow
- AI-native redesign
- Human steps
- Agent steps
- Shared steps
- Memory events
- Governance points
- Metrics
- Risks
- JSON workflow spec
- Example run

## Initial workflow coverage

- Hiring pipeline
- Sales pipeline
- Customer onboarding
- Sprint planning
- Pull request review
- Product discovery
- Incident response
- Budget planning
- Meeting management
- Knowledge management
- Campaign planning
- Vendor onboarding
