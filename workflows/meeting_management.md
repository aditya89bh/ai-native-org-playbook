# Meeting Management Workflow

## Current workflow

Meetings often create scattered notes, unclear decisions, weak follow-ups, and poor organizational memory.

## AI-native redesign

Agents support agenda preparation, context retrieval, decision capture, follow-up tracking, and meeting memory updates.

## Human steps

- Set meeting purpose
- Make decisions
- Resolve disagreements
- Own commitments
- Approve sensitive notes

## Agent steps

- Draft agenda
- Retrieve prior context
- Summarize decisions
- Track action items
- Update meeting memory

## Shared steps

- Pre-read preparation
- Meeting facilitation
- Decision review
- Follow-up tracking

## Memory events

- Agenda created
- Decision captured
- Action item assigned
- Follow-up status updated
- Meeting summary saved

## Governance points

Humans approve sensitive summaries, decisions, commitments, and externally shared notes.

## Metrics

- Decision clarity
- Follow-up completion
- Meeting load
- Context reuse
- Action item latency

## Risks

- Incorrect decisions captured
- Sensitive context over-shared
- Action ownership ambiguity
- Summary replacing alignment
