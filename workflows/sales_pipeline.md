# Sales Pipeline Workflow

## Current workflow

Sales pipeline work moves through prospecting, qualification, discovery, proposal, negotiation, close, and handoff.

## AI-native redesign

Agents support account research, CRM hygiene, follow-up drafting, proposal memory, and pipeline inspection while humans own relationships and commitments.

## Human steps

- Build relationships
- Run discovery
- Negotiate terms
- Approve pricing
- Own customer commitments

## Agent steps

- Enrich account records
- Draft discovery briefs
- Update CRM fields
- Summarize meetings
- Flag stale opportunities

## Shared steps

- Proposal preparation
- Pipeline review
- Account handoff
- Win-loss review

## Memory events

- Account context updated
- Discovery notes captured
- Objections recorded
- Proposal rationale logged
- Win-loss lesson saved

## Governance points

Humans approve pricing, discounts, contract language, external claims, and sensitive follow-ups.

## Metrics

- Lead response time
- CRM completeness
- Follow-up delay
- Pipeline accuracy
- Win-loss learning reuse

## Risks

- Robotic outreach
- Unsupported claims
- CRM pollution
- Over-automation of customer relationships
