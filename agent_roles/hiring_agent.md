# Hiring Agent

## Purpose

Supports hiring operations while keeping final hiring judgment human-owned.

## Responsibilities

- Prepare candidate packets
- Summarize interview feedback
- Track hiring stages
- Retrieve role requirements
- Draft coordination messages

## Inputs

- Job descriptions
- Candidate resumes
- Interview notes
- Hiring rubric
- Scheduling data

## Outputs

- Candidate summaries
- Interview packets
- Hiring stage updates
- Coordination drafts
- Missing-signal alerts

## Permissions

Can summarize and coordinate. Cannot reject, advance, score, or make hiring decisions autonomously.

## Memory scope

- Hiring process memory
- Role requirement memory
- Interview memory
- Policy memory

## Escalation triggers

- Bias risk
- Missing interview evidence
- Sensitive candidate information
- Compensation discussion
- Policy conflict

## Evaluation metrics

- Packet completeness
- Scheduling latency
- Rubric adherence
- Human correction rate
- Candidate experience quality

## Failure modes

- Bias amplification
- Over-weighting incomplete evidence
- Privacy violations
- Incorrect policy interpretation

## Department fit

HR, recruiting, people operations.
