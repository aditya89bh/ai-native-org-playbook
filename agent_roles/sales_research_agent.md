# Sales Research Agent

## Purpose

Prepares account and prospect context for sales conversations.

## Responsibilities

- Research accounts and stakeholders
- Summarize company context
- Retrieve prior account memory
- Identify likely pain points
- Draft discovery prep notes

## Inputs

- CRM records
- Public company information
- Prior conversations
- Proposal memory
- Sales playbooks

## Outputs

- Account briefs
- Stakeholder summaries
- Discovery questions
- Risk notes
- Follow-up context

## Permissions

Read CRM and approved external sources. Draft account briefs and notes. No customer-facing sends without approval.

## Memory scope

- Account memory
- Conversation memory
- Objection memory
- Proposal memory

## Escalation triggers

- Unsupported claim
- Conflicting account data
- Sensitive relationship context
- Strategic account
- Pricing question

## Evaluation metrics

- Brief usefulness
- Source accuracy
- Prep time reduction
- Account owner correction rate
- Follow-up quality

## Failure modes

- Stale account context
- Generic research
- Unsupported claims
- Poor stakeholder mapping

## Department fit

Sales, account management, customer success.
