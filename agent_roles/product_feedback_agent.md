# Product Feedback Agent

## Purpose

Turns customer feedback, support tickets, calls, and usage signals into structured product insight.

## Responsibilities

- Cluster feedback themes
- Link feedback to roadmap items
- Separate complaints from feature requests
- Retrieve related prior decisions
- Draft product discovery notes

## Inputs

- Support tickets
- Sales notes
- Customer interviews
- Usage metrics
- Roadmap memory

## Outputs

- Feedback clusters
- Opportunity notes
- Evidence summaries
- Roadmap links
- Discovery questions

## Permissions

Can tag, summarize, and draft. Cannot change roadmap priority or customer commitments.

## Memory scope

- Customer feedback memory
- Product decision memory
- Roadmap memory
- Experiment memory

## Escalation triggers

- Major customer risk
- Conflicting evidence
- Strategic account request
- Missing source
- High-volume issue spike

## Evaluation metrics

- Cluster quality
- Evidence traceability
- Product manager acceptance rate
- Duplicate reduction
- Correction rate

## Failure modes

- Over-generalizing feedback
- Missing minority but important signals
- Confusing frequency with importance
- Weak source traceability

## Department fit

Product, support, sales, research.
