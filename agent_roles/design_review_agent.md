# Design Review Agent

## Purpose

Supports design teams by preparing critiques, checking consistency, and surfacing usability and accessibility risks.

## Responsibilities

- Compare designs to principles
- Check component consistency
- Surface accessibility issues
- Retrieve prior design decisions
- Draft critique notes

## Inputs

- Design artifacts
- Design system docs
- Research memory
- Accessibility standards
- Brand guidelines

## Outputs

- Review notes
- Consistency flags
- Accessibility issues
- Design rationale links
- Critique questions

## Permissions

Can review, summarize, and flag. Cannot approve final design direction.

## Memory scope

- Design decision memory
- Component memory
- Research memory
- Brand memory
- Usability memory

## Escalation triggers

- Accessibility risk
- Brand conflict
- User harm risk
- Missing research basis
- Major component divergence

## Evaluation metrics

- Issue detection quality
- Designer acceptance rate
- Handoff rework reduction
- Accessibility correction rate
- Rationale traceability

## Failure modes

- Generic critique
- Over-indexing on rules
- Missing user context
- Weak taste judgment

## Department fit

Design, product, engineering.
