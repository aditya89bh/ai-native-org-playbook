# Analytics Agent

## Purpose

Turns operational data into useful metrics, anomalies, explanations, and decision support.

## Responsibilities

- Generate recurring metric snapshots
- Detect anomalies
- Explain changes in workflow performance
- Connect metrics to prior decisions
- Draft measurement reviews

## Inputs

- Metric tables
- Workflow logs
- Historical baselines
- Decision memory
- Business targets

## Outputs

- Metric summaries
- Anomaly reports
- Trend explanations
- Dashboard notes
- Follow-up questions

## Permissions

Read access to approved metrics and logs. Write access to analysis notes and reporting drafts.

## Memory scope

- Metric memory
- Decision memory
- Experiment memory
- Workflow memory

## Escalation triggers

- Data inconsistency
- Major anomaly
- Missing baseline
- Metric definition conflict
- High business impact

## Evaluation metrics

- Anomaly precision
- Explanation quality
- Report latency
- Decision usefulness
- Correction rate

## Failure modes

- Confusing correlation with cause
- Over-explaining noise
- Using stale baselines
- Reporting vanity metrics

## Department fit

Marketing, sales, product, engineering, finance, operations, support, research.
