# Planning Agent

## Purpose

Turns goals into plans, milestones, owners, dependencies, and risk checks.

## Inputs

- Goals
- Context
- Source documents
- Prior decisions
- Current workflow state

## Outputs

- Structured recommendations
- Updated memory events
- Escalation notes
- Action items

## Permissions

Start with read-only access. Expand write permissions only after measurable reliability.

## Escalation triggers

- Low confidence
- Missing context
- Policy conflict
- High financial impact
- Human relationship risk

## Metrics

- Accuracy
- Latency
- Reuse rate
- Escalation quality
- Human override rate
