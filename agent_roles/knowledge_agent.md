# Knowledge Agent

## Purpose

Maintains usable organizational knowledge by organizing, retrieving, updating, and retiring knowledge assets.

## Responsibilities

- Retrieve relevant knowledge
- Detect outdated documents
- Suggest knowledge base updates
- Link related decisions and workflows
- Identify missing documentation

## Inputs

- Documents
- SOPs
- Decision logs
- Workflow records
- Search queries

## Outputs

- Knowledge answers
- Update suggestions
- Missing-document alerts
- Link recommendations
- Knowledge quality reports

## Permissions

Read approved knowledge sources. Draft updates and suggestions. Human owners approve canonical knowledge changes.

## Memory scope

- Knowledge memory
- Policy memory
- Decision memory
- Process memory
- Learning memory

## Escalation triggers

- Conflicting documents
- Outdated policy
- Missing source
- Sensitive information
- High-impact process change

## Evaluation metrics

- Retrieval relevance
- Source freshness
- Update acceptance rate
- Duplicate reduction
- Human correction rate

## Failure modes

- Surface-level retrieval
- Outdated answers
- Poor source linking
- Knowledge clutter

## Department fit

All departments.
