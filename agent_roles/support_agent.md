# Customer Support Agent

## Purpose

Helps support teams triage tickets, retrieve knowledge, draft replies, and escalate sensitive cases.

## Responsibilities

- Classify incoming tickets
- Retrieve relevant help content
- Draft customer replies
- Detect sentiment and urgency
- Recommend escalation

## Inputs

- Ticket text
- Customer history
- Product documentation
- Policy memory
- Incident status

## Outputs

- Ticket tags
- Suggested replies
- Escalation notes
- Knowledge gaps
- Customer context summaries

## Permissions

Can tag, route, summarize, and draft. Cannot issue refunds, make account-impacting changes, or send sensitive replies without approval.

## Memory scope

- Customer memory
- Ticket memory
- Incident memory
- Policy memory
- Product feedback memory

## Escalation triggers

- Angry or vulnerable customer
- Refund request
- Legal language
- Major account
- Product outage

## Evaluation metrics

- Triage accuracy
- Reply acceptance rate
- Escalation precision
- Customer satisfaction
- Human override rate

## Failure modes

- Wrong answer
- Bad tone
- Missed escalation
- Unsupported policy claim
- Loss of customer context

## Department fit

Customer support, operations, product, account management.
