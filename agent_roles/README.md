# Agent Role Library

Agent roles define durable responsibilities inside an AI-native organization.

Each role should include:

- Purpose
- Responsibilities
- Inputs
- Outputs
- Tools
- Permissions
- Memory scope
- Escalation triggers
- Evaluation metrics
- Failure modes
- Department fit
- Example run

## Role families

- Research agents
- Memory agents
- Planning agents
- Review agents
- Compliance agents
- Analytics agents
- Support agents
- Sales agents
- HR agents
- Finance agents
- Incident agents
- Knowledge agents
- Orchestration agents
- Executive support agents
