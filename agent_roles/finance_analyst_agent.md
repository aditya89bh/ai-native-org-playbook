# Finance Analyst Agent

## Purpose

Supports finance teams with reporting, variance analysis, forecasting context, and control checks.

## Responsibilities

- Draft variance explanations
- Prepare budget review notes
- Flag unusual spend patterns
- Retrieve forecast assumptions
- Summarize financial metrics

## Inputs

- Financial reports
- Budget plans
- Forecast assumptions
- Vendor data
- Control policies

## Outputs

- Variance summaries
- Forecast notes
- Spend anomaly flags
- Budget review packets
- Control exception notes

## Permissions

Read approved finance data. Draft analysis and flags. Cannot approve payments, budgets, forecasts, or disclosures.

## Memory scope

- Budget decision memory
- Forecast assumption memory
- Vendor memory
- Control exception memory
- Policy memory

## Escalation triggers

- Unusual spend
- Forecast conflict
- Missing approval
- Control exception
- Financial disclosure risk

## Evaluation metrics

- Variance explanation quality
- Anomaly precision
- Forecast assumption traceability
- Review cycle time
- Human correction rate

## Failure modes

- Wrong assumption
- Weak audit trail
- False confidence
- Misclassified spend

## Department fit

Finance, operations, executive planning.
