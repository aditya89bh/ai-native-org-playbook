# Research Department Redesign

## Current operating model

Research teams gather external knowledge, user evidence, technical references, market signals, and strategic insight.

## AI-native redesign

AI-native research uses agents for source discovery, synthesis, citation tracking, hypothesis mapping, evidence grading, and decision support.

## Human-led work

- Research agenda
- Interpretation
- Strategic synthesis
- Ethical judgment
- Final recommendations

## Agent-assisted work

- Literature discovery
- Source comparison
- Evidence extraction
- Interview synthesis
- Research brief drafting

## Agent-executed work

- Citation formatting
- Source deduplication
- Research log updates
- Tagging
- Follow-up question generation

## Recommended agents

- Source Discovery Agent
- Evidence Synthesis Agent
- Citation Agent
- Interview Analysis Agent
- Research Memory Agent

## Core workflows

- Research brief creation
- Evidence review
- Interview synthesis
- Competitive research
- Strategic insight generation

## Memory map

- Source memory
- Evidence memory
- Hypothesis memory
- Interview memory
- Decision memory

## Governance

Humans approve conclusions, strategic recommendations, externally cited claims, and high-stakes interpretations.

## Metrics

- Research cycle time
- Source quality
- Evidence reuse
- Citation accuracy
- Decision impact
- Correction rate

## Risks

- Shallow synthesis
- Weak source quality
- Citation errors
- Confirmation bias
- Treating uncertain evidence as fact

## Maturity checklist

- Sources are traceable
- Evidence is graded
- Hypotheses are explicit
- Research memory is searchable
- Agents separate facts from interpretation
