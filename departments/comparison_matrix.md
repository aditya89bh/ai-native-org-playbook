# Department Comparison Matrix

| Department | Governance | Primary memory | Highest-value agent use |
| --- | --- | --- | --- |
| Sales | Medium | Account memory | Research, CRM hygiene, follow-up |
| Marketing | Medium | Campaign memory | Content operations and experiment analysis |
| Product | Medium | Product decision memory | Feedback synthesis and roadmap support |
| Engineering | High | Architecture and incident memory | Review, testing, incident support |
| Design | Medium | Design decision memory | Research synthesis and consistency checks |
| HR | High | Policy and employee process memory | Hiring operations and onboarding |
| Finance | High | Budget and control memory | Reporting, variance, control checks |
| Operations | Medium | Process and incident memory | Monitoring, SOPs, vendor coordination |
| Customer Support | Medium | Customer and ticket memory | Triage, reply drafts, escalation |
| Research | Medium | Source and evidence memory | Discovery, synthesis, citation tracking |
| Legal and Compliance | High | Contract and policy memory | Intake, clause review, obligation tracking |

## Interpretation

High-governance departments are not bad candidates. They simply require stronger approval thresholds, audit trails, source citation, and human accountability.
