# Department Playbooks

Department playbooks translate the framework library into specific functions.

Each department guide should cover:

- Current operating model
- AI-native redesign
- Human-led work
- Agent-assisted work
- Agent-executed work
- Recommended agent roles
- Core workflows
- Organizational memory map
- Governance requirements
- Metrics
- Risks
- Maturity checklist

## Department coverage

- Sales
- Marketing
- Product
- Engineering
- Design
- HR
- Finance
- Operations
- Customer support
- Research
- Legal and compliance
