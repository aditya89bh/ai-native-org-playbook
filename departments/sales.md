# Sales Department Redesign

## Purpose

Revenue workflows redesigned around prospect research, qualification, follow-up, CRM memory, and account planning.

## Human-led work

- Strategy
- Relationship-sensitive decisions
- High-risk approvals
- Judgment-heavy tradeoffs

## Agent-assisted work

- Research
- Summaries
- Drafting
- Task routing
- Knowledge retrieval
- Pattern detection

## Agent-executed work

- Low-risk updates
- Status reports
- Checklist validation
- Reminder generation
- Data normalization

## Required memory

- Decision memory
- Project memory
- Policy memory
- Learning memory

## Governance

Agents must operate with explicit permission boundaries, audit logs, and escalation triggers.
