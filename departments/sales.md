# Sales Department Redesign

## Current operating model

Sales teams manage prospecting, qualification, discovery, demos, proposals, negotiations, account plans, and CRM updates.

## AI-native redesign

AI-native sales uses agents for account research, CRM hygiene, call preparation, follow-up drafting, objection memory, and pipeline inspection.

## Human-led work

- Sales strategy
- Relationship building
- Discovery judgment
- Negotiation
- Enterprise account decisions

## Agent-assisted work

- Account research
- Call preparation
- Objection synthesis
- Proposal drafting
- Pipeline analysis

## Agent-executed work

- CRM field updates
- Meeting summaries
- Follow-up reminders
- Lead enrichment
- Duplicate detection

## Recommended agents

- Account Research Agent
- CRM Hygiene Agent
- Follow-Up Agent
- Pipeline Review Agent
- Customer Memory Agent

## Core workflows

- Prospect qualification
- Discovery preparation
- Proposal generation
- Pipeline review
- Account handoff

## Memory map

- Account memory
- Conversation memory
- Objection memory
- Proposal memory
- Decision memory

## Governance

Humans approve pricing, discounts, contractual commitments, customer promises, and sensitive relationship messages.

## Metrics

- Lead response time
- CRM completeness
- Meeting preparation time
- Pipeline accuracy
- Follow-up quality
- Win-loss learning reuse

## Risks

- Robotic outreach
- CRM pollution
- Unsupported claims
- Over-automation of relationships
- Outdated account context

## Maturity checklist

- CRM data is clean enough for agents
- Sales decisions are logged
- Pricing approvals are explicit
- Customer claims are source-supported
- Follow-up drafts are reviewed for tone
