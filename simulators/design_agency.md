# Design Agency Simulator

## Organization profile

A 40-person design agency using agents to support research, critique, project delivery, client communication, and knowledge reuse.

## Departments

- Design
- Research
- Sales
- Operations
- Finance
- Client delivery

## Agent roles

- Design Review Agent
- Research Agent
- Knowledge Agent
- Planning Agent
- Chief of Staff Agent
- Analytics Agent

## Core workflows

- Product discovery
- Meeting management
- Knowledge management
- Sales pipeline
- Budget planning

## Memory map

- Research memory
- Design decision memory
- Client memory
- Project memory
- Component memory

## Governance map

- Creative direction remains human-led
- Client-facing recommendations require lead review
- Research claims require source traceability
- Brand-sensitive outputs require approval

## Metrics

- Research reuse
- Design cycle time
- Handoff rework
- Client follow-up quality
- Proposal cycle time

## Before scenario

Design rationale is lost across critiques, client calls, research notes, and handoffs.

## After scenario

Agents preserve rationale, retrieve research, prepare critiques, and improve handoffs while designers keep taste and judgment.
