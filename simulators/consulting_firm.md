# Consulting Firm Simulator

## Organization profile

A 120-person consulting firm redesigning research, proposal, delivery, and knowledge reuse workflows around agents.

## Departments

- Sales
- Research
- Operations
- Finance
- HR
- Delivery teams

## Agent roles

- Research Agent
- Knowledge Agent
- Chief of Staff Agent
- Planning Agent
- Analytics Agent
- Compliance Agent

## Core workflows

- Sales pipeline
- Meeting management
- Knowledge management
- Budget planning
- Vendor onboarding

## Memory map

- Client memory
- Proposal memory
- Project memory
- Decision memory
- Knowledge memory

## Governance map

- Client-facing recommendations require partner review
- Pricing and commercial terms require approval
- Confidential client context is role restricted
- Case study reuse requires permission

## Metrics

- Proposal cycle time
- Research reuse
- Project margin visibility
- Knowledge freshness
- Client follow-up quality

## Before scenario

Teams repeatedly recreate research and lose context when consultants rotate across engagements.

## After scenario

Agents retrieve prior work, draft briefs, track decisions, and preserve project learning across client teams.
