# Manufacturing Company Simulator

## Organization profile

A mid-size manufacturing company using agents to improve operations, quality, maintenance, procurement, and customer support.

## Departments

- Operations
- Finance
- Customer support
- Engineering
- Legal and compliance
- HR

## Agent roles

- Workflow Orchestration Agent
- Incident Agent
- Knowledge Agent
- Finance Analyst Agent
- Compliance Agent
- Analytics Agent

## Core workflows

- Vendor onboarding
- Incident response
- Knowledge management
- Budget planning
- Customer onboarding

## Memory map

- Process memory
- Vendor memory
- Incident memory
- Policy memory
- Quality issue memory

## Governance map

- Production process changes require operations approval
- Vendor decisions require procurement and finance review
- Safety or quality incidents require strict audit
- Customer commitments require account owner approval

## Metrics

- Incident repeat rate
- Vendor onboarding time
- SOP freshness
- Quality issue recurrence
- Approval latency

## Before scenario

Operations knowledge is distributed across people, spreadsheets, SOPs, and incident notes.

## After scenario

Agents route exceptions, retrieve SOPs, preserve incident lessons, and highlight process risks before they repeat.
