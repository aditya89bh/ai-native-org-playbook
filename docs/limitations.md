# Limitations

This repository is a design and implementation toolkit, not a guarantee that an organization can safely automate work.

## Current limitations

- Scores are heuristic, not empirical benchmarks.
- Calculators are intended for structured thinking, not final decisions.
- Industry-specific compliance rules are not fully modeled.
- Human accountability must be defined before any real deployment.
- Memory policies must be adapted to legal, privacy, and security requirements.
- Simulators are illustrative examples, not exhaustive operating models.

## Safe use

Use this repository to structure discovery, design pilots, and build governance-aware workflows. Do not use it as a substitute for domain experts, legal review, security review, or operational judgment.
