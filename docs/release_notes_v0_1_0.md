# v0.1.0 Release Notes Draft

`v0.1.0` will mark the first usable release of the AI Native Organization Playbook.

## Highlights

- Practical framework for AI-native organization design
- Department playbooks across major business functions
- Agent role library with JSON specs
- Workflow library with JSON specs
- Memory and governance architecture
- Organization simulators
- CLI calculators
- Markdown report generation
- MkDocs documentation site

## Intended users

- Founders
- Consultants
- Operators
- Product and engineering leaders
- AI transformation teams

## Known limitations

- Scores are heuristic
- Simulators are illustrative
- Compliance rules require domain adaptation
- Human accountability is required for real deployment
