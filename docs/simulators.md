# Simulators and Examples

Phase 6 turns the playbook into concrete organization examples.

## Included simulators

- AI-native startup
- Consulting firm
- Manufacturing company
- University
- Design agency
- Robotics company

## CLI usage

```bash
ai-org-playbook simulator simulators/specs/robotics_company.json
```

The command returns:

- Simulator complexity
- Readiness hint

## Design rule

Simulators should connect departments, agents, workflows, memory, governance, and metrics into one operating model. A simulator is weak if it only describes an organization without showing how work changes.
