# Roadmap

The repository is planned as a 300 to 350 commit build.

## Phase 0: Foundation

Repository structure, core thesis, maturity model, CLI, tests, and documentation site.

## Phase 1: Framework library

Operating model, department redesign, agent responsibility, human-agent collaboration, memory, and governance models.

## Phase 2: Department playbooks

Guides for sales, marketing, product, engineering, design, HR, finance, support, operations, and research.

## Phase 3: Agent role library

Reusable specs for research, memory, planning, review, manager, compliance, analytics, support, hiring, and incident agents.

## Phase 4: Workflow library

Hiring, sales pipeline, onboarding, sprint planning, PR review, discovery, incident response, budget planning, meetings, and knowledge management.

## Phase 5: Memory and governance depth

Schemas, retention policies, audit models, permission models, escalation matrices, and risk tiers.

## Phase 6: Simulators and examples

Startup, consulting firm, manufacturing company, university, design agency, and robotics company examples.

## Phase 7: Interactive tools

Maturity calculator, opportunity scorer, staffing calculator, ROI calculator, transformation planner, and workflow simulator.

## Phase 8: Polish and release

Documentation site, examples, case studies, release checklist, and v1.0.0 tag.
