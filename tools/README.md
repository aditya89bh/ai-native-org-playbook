# Interactive Tools and Calculators

This section collects executable tools that turn the playbook into practical assessments.

## Tool categories

- Organization maturity calculator
- Readiness calculator
- Agent opportunity calculator
- Department complexity calculator
- Workflow surface-area calculator
- Governance risk calculator
- Memory policy calculator
- Simulator complexity calculator
- Transformation roadmap generator
- Markdown report exporter

## Design rule

A calculator is useful only if it changes a decision. Each tool should produce a score, interpretation, and recommended next step.
