# Example: Customer Support Department Redesign

## Candidate pilot

AI-assisted ticket triage and reply drafting.

## Current pain

Support teams lose time routing repetitive tickets and searching for the right policy or product answer.

## Agent roles

- Support Triage Agent
- Knowledge Retrieval Agent
- Reply Drafting Agent
- Escalation Agent

## Memory used

- Customer memory
- Ticket memory
- Policy memory
- Product feedback memory

## Governance

Agents can classify, route, and draft. Humans approve sensitive messages, refunds, major account responses, and policy exceptions.

## Baseline metrics

- First response time
- Resolution time
- Escalation accuracy
- Customer satisfaction

## Target outcome

Reduce triage load while improving response consistency and escalation quality.
