# Example: Incident Agent Run

## Workflow

Production incident response support.

## Input

- Incident report
- System logs
- Chat updates
- Runbook
- Prior incident records

## Agent actions

1. Build a timeline.
2. Retrieve similar incidents.
3. Track owners and unresolved actions.
4. Draft internal status updates.
5. Capture post-incident learning.

## Output

- Timeline
- Similar incident links
- Action tracker
- Status update draft
- Post-incident memory record

## Human review

Incident commander approves updates, resolution status, and final postmortem.

## Success metric

Faster situational awareness and better post-incident learning without weakening human ownership.
