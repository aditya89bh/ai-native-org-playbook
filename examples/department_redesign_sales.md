# Example: Sales Department Redesign

## Candidate pilot

AI-assisted account preparation for discovery calls.

## Current pain

Account executives spend too much time collecting scattered context before calls. CRM notes are incomplete and prior objections are hard to find.

## Agent roles

- Account Research Agent
- Customer Memory Agent
- Follow Up Agent

## Memory used

- Account memory
- Conversation memory
- Objection memory
- Proposal memory

## Governance

The agent may draft briefs and follow ups. Humans approve customer-facing messages and pricing language.

## Baseline metrics

- Preparation time per call
- CRM completeness
- Follow up delay
- Account executive satisfaction

## Target outcome

Reduce preparation time while improving context quality and follow up consistency.
