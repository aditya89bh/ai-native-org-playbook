# Example: Robotics Company Simulator Run

## Scenario

A customer deployment reports repeated robot task failures after a field update.

## Agents involved

- Incident Agent
- Product Feedback Agent
- Knowledge Agent
- Workflow Orchestration Agent

## Run

1. Incident Agent builds a timeline from support notes and logs.
2. Knowledge Agent retrieves similar field incidents and architecture decisions.
3. Product Feedback Agent links the issue to roadmap and customer feedback memory.
4. Workflow Orchestration Agent routes engineering, support, and product follow-ups.
5. Human owners approve customer communication and engineering changes.

## Output

- Field incident timeline
- Similar incident references
- Product feedback link
- Engineering follow-up
- Customer update draft

## Success metric

The company learns from field issues faster and reduces repeated deployment failures.
