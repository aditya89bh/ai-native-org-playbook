# Example: Customer Onboarding Workflow Run

## Workflow

Customer onboarding for a new B2B account.

## Agents involved

- Customer Memory Agent
- Workflow Orchestration Agent
- Knowledge Agent

## Run

1. Agent drafts onboarding plan from kickoff notes.
2. Human customer owner reviews goals and commitments.
3. Agent tracks checklist completion.
4. Agent flags unresolved risks.
5. Human approves success plan.
6. Agent creates handoff memory for support.

## Output

- Onboarding plan
- Risk list
- Checklist state
- Handoff memory

## Success metric

Lower time to value and cleaner handoff to support.
