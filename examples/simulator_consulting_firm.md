# Example: Consulting Firm Simulator Run

## Scenario

A partner team is preparing a proposal for a repeat client but prior project learning is scattered.

## Agents involved

- Research Agent
- Knowledge Agent
- Chief of Staff Agent
- Analytics Agent

## Run

1. Knowledge Agent retrieves prior engagement memory.
2. Research Agent prepares market and client context.
3. Analytics Agent summarizes margin and delivery patterns from similar work.
4. Chief of Staff Agent drafts the proposal preparation brief.
5. Partner reviews commercial claims and client-facing recommendations.

## Output

- Client context brief
- Prior project lessons
- Proposal risk notes
- Follow-up tracker

## Success metric

Faster proposal creation with better reuse of institutional knowledge.
