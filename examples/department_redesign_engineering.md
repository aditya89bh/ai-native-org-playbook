# Example: Engineering Department Redesign

## Candidate pilot

AI-assisted pull request review preparation.

## Current pain

Reviewers spend time reconstructing intent, checking related issues, finding prior decisions, and identifying missing tests.

## Agent roles

- Issue Triage Agent
- Code Review Agent
- Test Generation Agent
- Incident Memory Agent

## Memory used

- Architecture decision memory
- Code review memory
- Incident memory
- Technical debt memory

## Governance

Agents can summarize, suggest tests, and flag risks. Humans approve code changes, merges, and production releases.

## Baseline metrics

- Review cycle time
- Defect escape rate
- Test gap rate
- Reviewer satisfaction

## Target outcome

Improve review quality and reduce reviewer load without weakening engineering ownership.
