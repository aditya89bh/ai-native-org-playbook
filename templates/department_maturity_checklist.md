# Department Maturity Checklist

## Workflow clarity

- Core workflows are documented
- Owners are explicit
- Handoffs are visible
- Failure points are known

## Agent readiness

- Candidate agent roles are defined
- Inputs and outputs are known
- Permissions are bounded
- Evaluation metrics exist

## Memory readiness

- Useful memory types are identified
- Retention rules are defined
- Correction path exists
- Sensitive memory is scoped

## Governance readiness

- Approval thresholds exist
- Escalation triggers exist
- Audit requirements are known
- Human accountability is explicit

## Measurement readiness

- Current baseline is measured
- Target improvement is defined
- Review cadence exists
- Outcome metrics are not confused with usage metrics
