# Approval Matrix

Approval matrices define which actions agents can take and which require human review.

## Approval dimensions

- Financial impact
- Customer impact
- Employee impact
- Legal exposure
- Reversibility
- Public visibility
- Confidence
- Novelty

## Approval modes

| Mode | Use |
| --- | --- |
| No approval | Low-risk reversible work |
| Batch review | Repetitive low-risk work |
| Sampled review | Mature workflows with measured quality |
| Pre-approval | Consequential or sensitive actions |
| Dual approval | High-risk financial, legal, or people decisions |

## Design rule

Do not use the same approval rule for every department. Approval should match risk and reversibility.
