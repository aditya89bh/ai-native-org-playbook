# Escalation Matrix

Escalation defines when an agent must stop, ask, route, or hand off work.

## Trigger categories

1. Low confidence
2. Missing context
3. Policy conflict
4. Sensitive stakeholder
5. High financial impact
6. Legal or compliance exposure
7. Novel situation
8. Repeated failure

## Escalation levels

| Level | Action |
| --- | --- |
| L0 | Continue automatically |
| L1 | Add note for later review |
| L2 | Ask human before action |
| L3 | Route to accountable owner |
| L4 | Stop workflow until reviewed |

## Design rule

Escalation should be specific enough to protect the organization but not so broad that every useful workflow gets blocked.
