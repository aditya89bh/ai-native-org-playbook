# AI-Native Organization Maturity Model

## Level 1: Ad Hoc

AI usage is individual, inconsistent, and undocumented.

## Level 2: Assisted

Teams use AI tools for drafts, summaries, and basic support, but workflows are still human-designed and human-operated.

## Level 3: Augmented

AI participates in defined workflow steps. Some memory, review, and escalation rules exist.

## Level 4: Orchestrated

Agents coordinate across workflows with explicit permissions, logs, memory, and human oversight.

## Level 5: AI-Native

The organization treats humans, agents, memory, governance, and metrics as one operating system.

## Capabilities scored

1. Workflow clarity
2. Knowledge quality
3. Agent readiness
4. Governance
5. Memory design
6. Measurement
7. Change capacity
