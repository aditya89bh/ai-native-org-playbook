# AI-Native Readiness Assessment

Readiness measures whether the organization can safely benefit from agents.

## Capability areas

1. Workflow clarity
2. Data quality
3. Documentation maturity
4. Governance maturity
5. Tool access control
6. Memory architecture
7. Human review capacity
8. Measurement discipline
9. Change management

## Scoring

Score each area from 0 to 5.

- 0: absent
- 1: informal
- 2: partially documented
- 3: repeatable
- 4: measured
- 5: optimized

## Interpretation

Low readiness does not mean no AI. It means start with assistive, low-risk workflows before autonomous workflows.
