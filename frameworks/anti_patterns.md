# AI-Native Organization Anti-Patterns

## Chatbot wrapping

Adding a chatbot to a broken workflow without redesigning the workflow.

## Prompt theater

Treating prompt collections as transformation while ignoring permissions, memory, metrics, and ownership.

## Invisible agents

Deploying agents without clear owners, logs, or accountability.

## Automation-first thinking

Trying to replace humans before understanding decision rights and failure modes.

## Memory hoarding

Saving everything without retrieval, retention, correction, or forgetting.

## Governance paralysis

Adding so much approval that agents cannot create leverage.

## Metric blindness

Counting usage instead of measuring better work.
