# Value Measurement Framework

AI-native transformation must be measured as operating leverage, not novelty.

## Value categories

1. Speed
2. Quality
3. Cost
4. Risk reduction
5. Knowledge reuse
6. Human capacity
7. Customer experience
8. Learning rate

## Baseline first

Before a pilot, capture current cycle time, effort, error rate, cost, rework, and satisfaction.

## After measurement

Measure the same metrics after the agent-supported workflow is deployed.

## Warning

Do not count usage as value. A heavily used agent that does not improve work is still waste.
