# Organizational Memory Model

Organizational memory is the system that lets an organization learn from prior work.

## Memory objects

- Decisions
- Meetings
- Projects
- Customers
- Incidents
- Policies
- Experiments
- Lessons

## Memory lifecycle

1. Capture
2. Classify
3. Retrieve
4. Apply
5. Correct
6. Forget

## Quality criteria

Useful memory is source-linked, time-stamped, scoped, retrievable, correctable, and expire-able.

## Failure modes

- Everything is stored but nothing is retrieved
- Outdated memory keeps influencing work
- Private judgment becomes treated as fact
- Agents retrieve memories without confidence or context
