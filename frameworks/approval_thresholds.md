# Approval Thresholds

Approval thresholds define when agent work needs human authorization.

## Dimensions

- Financial impact
- Customer impact
- Employee impact
- Reversibility
- Public visibility
- Compliance exposure
- Novelty
- Confidence

## Default rule

The more irreversible, visible, sensitive, or ambiguous the action is, the more human approval is required.

## Approval modes

1. No approval
2. Batch review
3. Sampled review
4. Pre-approval
5. Dual approval
6. Executive approval

## Anti-pattern

Do not require approval for every agent action. That turns agents into slower interns instead of operational leverage.
