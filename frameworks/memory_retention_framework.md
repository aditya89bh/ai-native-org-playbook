# Memory Retention Framework

Not all organizational memory should live forever.

## Retention tiers

| Tier | Example | Default retention |
| --- | --- | --- |
| Operational | Daily task updates | 30 to 90 days |
| Project | Milestones and blockers | Project lifetime plus review window |
| Decision | Important tradeoffs and approvals | Long-term |
| Policy | Rules and constraints | Until superseded |
| Incident | Failure records and remediation | Long-term with review |
| Personal | Individual preferences or sensitive context | Strictly scoped and minimized |

## Retention questions

- Who will reuse this memory?
- What decision will it improve?
- What harm can stale memory cause?
- When should it expire?
- Who can correct it?
