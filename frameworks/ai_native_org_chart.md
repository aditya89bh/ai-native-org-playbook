# AI-Native Org Chart

AI-native org charts include both human reporting lines and agent operating lines.

## Human reporting lines

Human reporting lines define accountability, coaching, prioritization, and employment responsibility.

## Agent operating lines

Agent operating lines define which workflows an agent supports, which humans supervise it, and which systems it can access.

## Hybrid structure

A mature AI-native org chart shows:

- Human owner
- Agent role
- Workflow supported
- Permission tier
- Escalation owner
- Memory scope
- Evaluation metric

## Design risk

If agents are invisible in the org chart, accountability becomes invisible too.
