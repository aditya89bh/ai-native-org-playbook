# Human-Agent RACI

Use RACI to clarify responsibility when humans and agents share work.

## Definitions

- Responsible: does the work
- Accountable: owns the outcome
- Consulted: provides input
- Informed: receives updates

## AI-native adaptation

Agents can be responsible for bounded work, but humans remain accountable for consequential outcomes.

## Example

| Activity | Human owner | Agent | Reviewer | Stakeholder |
| --- | --- | --- | --- | --- |
| Research options | Accountable | Responsible | Consulted | Informed |
| Draft recommendation | Accountable | Responsible | Consulted | Informed |
| Final decision | Responsible | Consulted | Consulted | Informed |
| Log memory | Accountable | Responsible | Consulted | Informed |

## Rule

Never leave accountability with an agent. Accountability belongs to a human or governance body.
