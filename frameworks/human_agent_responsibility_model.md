# Human-Agent Responsibility Model

This model separates work by judgment, risk, reversibility, and context sensitivity.

## Responsibility bands

| Band | Owner | Description |
| --- | --- | --- |
| Observe | Agent | Agent watches signals and gathers context |
| Draft | Agent | Agent creates a first pass for human review |
| Recommend | Agent | Agent proposes options with evidence |
| Approve | Human | Human accepts, rejects, or modifies |
| Execute | Agent or human | Actor performs the approved action |
| Audit | Human and agent | System records and reviews outcomes |

## Human-owned work

Humans should own high-context, high-risk, relationship-sensitive, ambiguous, or irreversible decisions.

## Agent-owned work

Agents should own repetitive, well-bounded, measurable, reversible, and information-heavy work.

## Shared work

Most valuable workflows are shared. Agents increase throughput and memory. Humans preserve judgment and accountability.
